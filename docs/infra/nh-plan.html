Deployment for New Hampshire dealerships. Keep voice + CRM context local with a hardened egress path.

← Back to Overview Install Guide

DNS Approach

Pilot: Cloudflare in DNS-only (gray cloud) for voice.quirkcars.com (A → public IP)
Later: registrar DNS or self-hosted NS if preferred
CAA: allow letsencrypt.org for TLS issuance

Firewall & TLS

Inbound: TCP 80/443 → Caddy only
Outbound allow-list: Twilio, OpenAI, Let's Encrypt
TLS terminates at Caddy (Let’s Encrypt)

Replace Cloudflare?

Yes, if you move to registrar/self-host DNS.
If leaving Cloudflare entirely, consider WAF/DDoS alternatives (Nginx+ModSecurity, ISP scrubbing, etc.)
