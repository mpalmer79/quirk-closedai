Prompt Governance

• All prompts live in version control.
• Changes via PR with reviewer from Ops + IT.
• Tests: tone, compliance, and redaction pass before merge.

Data Handling

• On premise isolation. No raw audio stored.
• Logs redact names, phones, emails by default.
• Role-based access for transcripts and config.

Change Log — PR Template

Title: [Dept] Short description
Why: business outcome / defect
Risk: low/med/high
Test: steps & results
Rollback: how to revert
